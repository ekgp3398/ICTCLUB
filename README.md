# ICTCLUB

ICT융합공학부생들을 위한 커뮤니티

- 기능
  1. 회원관리(회원가입, 로그인, 로그아웃)
  2. 동아리별 커뮤니티 게시판
  3. ICT융합공학부 공지사항 확인
  4. 강남대학교 학사일정 확인
  5. 나에게 맞는 동아리는? 테스트 지원
